# ## Windows server 2008 AD证书服务安装过程

### Requirement
在企业内部常常需要用到证书服务，网络设备或者邮件系统通常可以设置使用证书进行认证，这里我们在AD上搭建一个证书服务器，作为企业内部的一个证书颁发机构。
### Content
下面是在Windows server 2008证书安装过程的详细步骤：
1. 添加证书服务角色
